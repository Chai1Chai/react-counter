# Counter App

A simple React counter application built with React Hooks.  
The app allows the user to increment and decrement a number.

---

## 🚀 Features

- Increase counter by 1
- Decrease counter by 1
- State management using `useState`
- Dynamic UI updates
- Styled with SCSS

---

## 🛠 Technologies

- React
- JavaScript (ES6)
- SCSS

---

## 📂 Project Structure

- `App.js` — main component with state logic  
- `index.js` — entry point  
- `index.scss` — styles  

---
